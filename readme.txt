A high-performance link checker for WordPress with affiliate detection and background processing.

== Description ==

LinkChecker is a powerful tool designed to help WordPress site owners maintain their SEO and user experience by identifying broken links, redirects, and affiliate patterns.

Key Features:
*   **High Performance**: Uses background processing via Action Scheduler to scan links without slowing down your site.
*   **Affiliate Detection**: Automatically identifies affiliate links using custom patterns.
*   **Real-time Dashboard**: Modern React-based dashboard for managing scans and viewing results.
*   **Flexible Scaling**: Configurable batch sizes and request delays to fit your server's resources.

== Installation ==

1. Upload the `flavor-link-checker` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the dashboard via 'Link Checker' in the admin sidebar.

== Frequently Asked Questions ==

You can trigger scans manually from the dashboard, or set up automated background checks.
